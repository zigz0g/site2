Frequently Asked Questions

    What are the controls for the GBA Section?

    A - X, J
    B - Z, Q, Y
    Select - Shift
    Start - Enter
    Right - ArrowRight
    Left - ArrowLeft
    Up - ArrowUp
    Down - ArrowDown
    R - 2
    L - 1

Saving

Saving is not guaranteed to work properly for all games. If this site were to get blocked, I would have to make a new one and your data would be gone. If you are using Super Mario 64 or a GBA game, make sure to press save before you exit.
